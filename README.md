# Trading-Journal

A mobile-friendly trading journal built as a single HTML file that runs in your browser with no installation required.
It covers two workflows in one app — live trade review and backtesting — designed around a Plan → Execute → Review daily loop for ES, NQ, and YM futures traders.
Live trade review walks you through a structured flowchart after each trade, classifying it as a good win, bad win, good loss, bad loss, good pass, or bad pass — forcing honest process evaluation over outcome bias. Each classification leads to three targeted reflection questions that get saved to your journal.
Backtesting lets you log each replayed trade with session, instrument, direction, daily bias, daily profile, entry/target/stop prices, R:R (auto-calculated), outcome, screenshot reference, and free-form notes.
Search filters both logs by instrument, outcome, session, date range, and keyword — with match highlighting.
Stats tracks good process rate, win rate, outcome breakdown, instrument usage, selection bias (weakest vs strongest), and backtest performance by session and daily profile.
Google Drive sync keeps your data backed up and accessible across devices. Everything works offline first, with local storage as the foundation.
Light and dark mode included, with a one-tap toggle.
